# Changelog

All notable changes to this project will be documented in this file.

## [1.0.2] - 2025-11-30
### Added
- [Burst Kalman Filter] Jobs and ECS ready now.

### Added
- [Value Adapter] customizable value processor, supporting any unmanaged types
